# Start Zoom Video SDK

1. Run `yarn install`
2. Add your SDK key and secret to a file named `.env` modeled after `.env.example`

```.env
SKIP_PREFLIGHT_CHECK=true
REACT_APP_ZOOM_SDK_KEY=""
REACT_APP_ZOOM_SDK_SECRET=""
```

3. Run `yarn start`
